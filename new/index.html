From Games to Cybersecurity

  1. I started using computers purely for gaming, fascinated by what software could do. Curiosity quickly evolved into creation when Java became my first programming language.

  2. As a teenager, I explored Android development, experimented with Unity and built small games using C#. Every project taught me how software is designed, built and improved.

  3. Over time my interests matured toward Linux, networking and cybersecurity. I explored countless Linux distributions, built personal tools, experimented with servers, automation and system internals, and gradually found my place in the cybersecurity world.

  4. Today I enjoy building tools that solve real problems, analyzing Formula 1 telemetry data, exploring networks, automating workflows and continuously learning how systems operate beneath the surface.
